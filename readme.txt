The Purplepass Ticketing plugin for Modern Tribe's Event Calendar allows you to add a robust ticketing system directly within your Wordpress website.

== Description ==

The Purplepass Ticketing plugin for Modern Tribe's Event Calendar allows you to add a robust ticketing system directly within your Wordpress website.

== Installation ==

Install the Purplepass plugin for The Event Calendar to your WordPress site, activate it, then follow the steps to link it to your existing Purplepass account. This account will be used for creation and management of your events.
